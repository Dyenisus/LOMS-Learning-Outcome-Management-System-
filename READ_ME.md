In order to run this project you need any type of env that has the specified packages in the requirements.txt

To use it with Virtual env run:

  Windows

    python -m venv .venv
    .venv\Scripts\activate
    pip install --upgrade pip
    pip install -r requirements.txt


  Macos

    python -m venv .venv
    source .venv/bin/activate
    pip install --upgrade pip
    pip install -r requirements.txt



feel free to copy and paste according to your os and then open the local server by using:

  Windows or Macos

    python manage.py runserver


and enjoy testing :D
